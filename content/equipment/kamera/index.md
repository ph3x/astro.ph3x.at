Die **Canon EOS 1000D** (in Nordamerika als *EOS Rebel XS*, in Japan als *EOS Kiss F* bekannt) ist eine digitale Spiegelreflexkamera, die ich unmodifiziert für die Astrofotografie einsetze.

### Technische Daten

| Eigenschaft | Wert |
|---|---|
| Sensor | APS-C CMOS, 22,2 × 14,8 mm |
| Auflösung | 10,1 Megapixel |
| Crop-Faktor | 1,6× |
| Prozessor | DIGIC III |
| ISO-Bereich | 100 – 1600 |
| Verschlusszeiten | 30 s – 1/4000 s |
| Serienbildrate | 3 Bilder/s |
| Display | 2,5″ TFT, 230.000 Punkte |
| Anschluss | Canon EF / EF-S |
| Abmessungen | 126 × 98 × 65 mm |
| Gewicht | 450 g (ohne Akku) |

### Einsatz in der Astrofotografie

Die Kamera wird **unmodifiziert** verwendet, d.h. der originale Infrarot-Sperrfilter ist noch eingebaut. Dadurch werden H-α-Emissionen (659 nm) teilweise herausgefiltert, was die Aufnahme von roten Emissionsnebeln einschränkt. Für Mond- und Planetenaufnahmen sowie Weitfeldaufnahmen ist sie jedoch sehr gut geeignet.
