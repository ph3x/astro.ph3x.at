Die **Sky-Watcher EQ6-R Pro** ist eine computergestützte parallaktische GoTo-Montierung und das Herzstück meines Aufbaus. Dank riemengetriebenem Antrieb und hoher Tragkraft eignet sie sich hervorragend für die Langzeitbelichtungs-Astrofotografie.

### Technische Daten

| Eigenschaft | Wert |
|---|---|
| Typ | Parallaktische GoTo-Montierung |
| Tragkraft | 20 kg |
| Eigengewicht (Kopf) | 17,2 kg |
| Antrieb | Riemengetriebe, Schrittmotoren |
| Schrittwinkel | 1,8° / 64 Mikroschritte |
| Schwenkgeschwindigkeit | bis 3,4°/s |
| Steuerung | SynScan GoTo (42.900+ Objekte) |
| Anschlüsse | ST-4 Autoguider, USB-B, PC-Direct |
| Stativ | 2″ Edelstahl |
| Gegengewichte | 2 × 5 kg |
| Sattelplatte | Dual D/V-Schiene |

### Einsatz in der Astrofotografie

Die riemengetriebene Mechanik sorgt für eine sehr ruhige und vibrationsarme Nachführung — ideal für Langzeitbelichtungen. Der integrierte **ST-4 Autoguider-Anschluss** ermöglicht die Verwendung eines Leitrohr-Systems zur präzisen Korrekturnachführung. Die Steuerung erfolgt über den SynScan-Controller oder per App via WLAN-Adapter.
